# College Teacher Rating - Колледж Оқытушыларды Бағалау Сайты

Оқытушыларды бағалауға арналған веб-қосымша.

## 🌐 Мүмкіндіктер

- Оқытушылар тізімін көру
- Іздеу және сүзу
- Жұлдызшамен бағалау (1-5)
- Пікір қалдыру
- Тіркелу/Кіру жүйесі
- Таңдаулы оқытушылар
- Әкімші панелі

## 🛠 Технологиялар

- React.js + Vite
- Supabase (PostgreSQL)
- React Router DOM
